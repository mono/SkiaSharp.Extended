# SKBlurHash

A **BlurHash** is a compact representation of a placeholder for an image.

For example, the SkiaSharp logo can be reduced to a tiny string:

```
LjPsbRxG%gx^aJxuM|W=?^X8Mxn$
```

| Original (79.7 KB) | BlurHash (36 bytes) |
| :----------------: | :-----------------: |
| ![][orig-img]      | ![][blur-img]       |

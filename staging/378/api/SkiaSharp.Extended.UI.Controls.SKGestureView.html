Remarks

Use this control when you want gesture-driven pan/zoom over custom-drawn content. For deep zoom image support, see SKDeepZoomView in the SkiaSharp.Extended.UI.Maui.DeepZoom package.

Example usage:

var gestureView = new SKGestureView();
gestureView.PaintSurface += (s, e) =>
{
    var canvas = e.Surface.Canvas;
    canvas.Clear(SKColors.White);
    canvas.SetMatrix(gestureView.GestureTracker.GetMatrix());
    // draw your content here
};

Initializes a new SKGestureView with the specified gesture options.

Properties

GestureTracker

The underlying gesture tracker. Use this to configure gesture options, subscribe to individual gesture events (tap, long press, etc.), or read the current transform.

Methods

Dispose()

Events

PaintSurface

Raised when the surface needs to be painted. Handle this to draw your content. Use Matrix to apply the current pan/zoom transform.

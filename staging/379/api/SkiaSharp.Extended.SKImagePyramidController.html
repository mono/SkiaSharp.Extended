Remarks

This class handles only tile and rendering concerns. Animation, spring physics, and gesture recognition belong in the consuming layer.

Typical usage:

  1. Call Load(SKImagePyramidDziSource, ISKImagePyramidTileFetcher) to load an image.
  2. Call SetControlSize(double, double) when the canvas size changes.
  3. Call Update() to schedule tile loads each frame.
  4. Call Render(ISKImagePyramidRenderer) from your canvas paint callback, passing the renderer for this frame.

Constructors

SKImagePyramidController(ISKImagePyramidTileCache?, int)

Initializes a new SKImagePyramidController with an optional custom cache. When cache is null, a default SKImagePyramidMemoryTileCache is used.

Properties

AspectRatio

The aspect ratio of the loaded image (width/height). 0 if not loaded.

Cache

The tile cache.

EnableLodBlending

Whether to draw lower-resolution fallback tiles while hi-res tiles are loading (LOD blending). Default is true.

IsIdle

Whether the controller has no pending tile loads.

NativeZoom

The zoom level at which one image pixel maps to exactly one screen pixel. Returns 0 if no image is loaded.

PendingTileCount

Number of tile fetches currently in flight.

SubImages

The sub-images from the loaded DZC, or empty if not loaded from a DZC.

TileLayout

The tile layout calculator.

TileSource

The loaded tile source, or null if not loaded.

Viewport

The current viewport.

Methods

Dispose()
GetZoomRect()

Returns the logical rectangle visible at the current viewport state.

Load(ISKImagePyramidSource, ISKImagePyramidTileFetcher)

Loads a DZI or IIIF tile source. Resets the viewport to show the full image.

Load(SKImagePyramidDziCollectionSource, ISKImagePyramidTileFetcher)

Loads a DZC collection source. Populates SubImages with the items in the collection. Does NOT set a renderable tile source — listen for CollectionOpenSucceeded and then call Load(ISKImagePyramidSource, ISKImagePyramidTileFetcher) with a specific sub-image source to actually render.

Pan(double, double)

Pans by the given screen-space delta.

Render(ISKImagePyramidRenderer)

Executes the two-pass LOD rendering pipeline using the provided renderer. The renderer is transient — it is only used during this call and should not be stored by the controller. This allows the renderer (and its canvas) to be created and discarded per frame, which is required by some graphics APIs.

ReplaceCache(ISKImagePyramidTileCache)

Replaces the active tile cache without disturbing the loaded image, viewport state, or zoom level. All pending tile loads are cancelled and in-flight tiles are discarded; the new cache starts empty and tiles are re-fetched as the view is repainted.

ResetView()

Resets the viewport to show the entire image.

SetControlSize(double, double)

Sets the control (canvas) size. Refits the viewport when the user has not manually zoomed.

SetViewport(double, double, double)

Sets the viewport directly and constrains it.

SetZoom(double)

Sets an absolute zoom level (1.0 = image fills the control width).

Update()

Schedules loading for visible tiles. Call from your render loop on every frame.

ZoomAboutLogicalPoint(double, double, double)

Zooms about a logical point. Factor > 1 zooms in, < 1 zooms out.

ZoomAboutScreenPoint(double, double, double)

Zooms about a screen-space point.

Events

CollectionOpenSucceeded

Fired when a DZC collection is successfully loaded and SubImages is populated. Use this event to know when the collection is ready. To render a specific image from the collection, call Load(ISKImagePyramidSource, ISKImagePyramidTileFetcher) with one of the sub-image sources from SubImages.

ImageOpenFailed

Fired when the image source fails to load.

ImageOpenSucceeded

Fired when the image source is loaded successfully.

InvalidateRequired

Fired when new tiles are loaded and the view needs repainting.

TileFailed

Fired when a tile fails to load.

ViewportChanged

Fired when the viewport position or zoom level changes.

EnableLodBlending

Whether to draw lower-resolution fallback tiles while hi-res tiles are loading (LOD blending). Default is true.

public bool EnableLodBlending { get; set; }

Property Value

bool

Remarks

Enabled (default): blurry ancestor tiles fill the screen immediately as you zoom in; they sharpen progressively. Best for interactive use.

Disabled: missing tiles show as blank until loaded. Preferred for scientific/medical imaging where blurry placeholders could be misleading.

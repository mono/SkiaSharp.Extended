Class SKDeepZoomTileCache

Namespace
SkiaSharp.Extended.DeepZoom

LRU cache for decoded tile bitmaps. Evicted bitmaps are deferred for disposal to avoid race conditions with the renderer (which may hold references to recently-returned bitmaps). Call FlushEvicted() at the start of each render frame.

FlushEvicted()

Disposes all bitmaps that were evicted since the last call. Call this at the start of each render frame (on the UI thread) to safely dispose bitmaps that are no longer in use.

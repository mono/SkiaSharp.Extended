Remarks

This class handles only the tile and rendering concerns. Animation (spring physics, easing) and gesture recognition belong in the consuming layer (e.g. a MAUI view).

Typical usage:

  1. Call Load(SKDeepZoomImageSource, ISKDeepZoomTileFetcher) to load an image.
  2. Call SetControlSize(double, double) when the canvas size changes.
  3. Call SetViewport(double, double, double) / Pan(double, double) / ZoomAboutScreenPoint(double, double, double) to navigate.
  4. Call Update() and Render(SKCanvas) from your render loop.

Methods

Dispose()
GetZoomRect()

Returns the logical rectangle visible at the current viewport state.

Load(SKDeepZoomCollectionSource, ISKDeepZoomTileFetcher)

Loads a DZC tile source, populates SubImages, and sets up the tile fetcher.

Load(SKDeepZoomImageSource, ISKDeepZoomTileFetcher)

Loads a DZI tile source and sets up the tile fetcher. Resets the viewport to show the full image.

Pan(double, double)

Pans by the given screen-space delta.

Render(SKCanvas)

Renders the current viewport state onto the given canvas.

ResetView()

Resets the viewport to show the entire image and clears the manual-zoom flag.

SetControlSize(double, double)

Sets the control (canvas) size. Call whenever the canvas is resized. When a source is loaded and the user has not manually zoomed, automatically refits the viewport so the image remains fully visible in the new size.

SetViewport(double, double, double)

Sets the viewport directly to the given state and constrains it.

SetZoom(double)

Sets an absolute zoom level (1.0 = image fills the control width). Zooms about the center of the control.

Update()

Schedules loading for visible tiles and returns whether any tiles are still pending. Call from your render loop on every frame.

ZoomAboutLogicalPoint(double, double, double)

Zooms about a logical point. Factor > 1 zooms in, < 1 zooms out.

ZoomAboutScreenPoint(double, double, double)

Zooms about a screen-space point. Factor > 1 zooms in, < 1 zooms out.

Events

ImageOpenSucceeded

InvalidateRequired

TileFailed

ViewportChanged

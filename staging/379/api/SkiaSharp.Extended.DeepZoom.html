Table of Contents

Namespace SkiaSharp.Extended.DeepZoom

Classes

SKDeepZoomBitmapTile

A ISKDeepZoomTile backed by an SKBitmap. This is the SkiaSharp-specific implementation of the opaque tile interface.

SKDeepZoomBitmapTileDecoder

Decodes image streams into SKDeepZoomBitmapTile using SkiaSharp.

SKDeepZoomCollectionSource

Represents a Deep Zoom Collection (DZC) — a composite tiled pyramid of multiple images. Parses the DZC XML descriptor and provides access to sub-images and composite tile math.

SKDeepZoomCollectionSubImage

Represents a single sub-image within a DZC collection. Maps to Silverlight's MultiScaleSubImage.

SKDeepZoomController

Orchestrates the Deep Zoom rendering pipeline: viewport management, tile scheduling, cache management, and rendering.

SKDeepZoomFileTileFetcher

Fetches tiles from the local file system and decodes them using a provided ISKDeepZoomTileDecoder. Accepts both plain file paths and file:// URIs.

SKDeepZoomHttpTileFetcher

Fetches tiles over HTTP and decodes them using a provided ISKDeepZoomTileDecoder. Thread-safe and reusable.

SKDeepZoomImageSource

Represents a Deep Zoom Image (DZI) tile source. Parses the DZI XML descriptor and provides tile pyramid math for computing tile URLs and dimensions.

SKDeepZoomMemoryTileCache

LRU cache for decoded tile images. Evicted tiles are deferred for disposal to avoid race conditions with the renderer. Call FlushEvicted() at the start of each render frame.

SKDeepZoomRenderer

SkiaSharp implementation of ISKDeepZoomRenderer. Draws Deep Zoom tiles onto an SKCanvas.

SKDeepZoomSubImage

Represents a single sub-image within a DZC collection, with its position in the mosaic. Mirrors Silverlight's MultiScaleSubImage with inverted viewport coordinates.

SKDeepZoomTileFailedEventArgs

Event args for when a tile fails to load.

SKDeepZoomTileLayout

Computes tile geometry for the Deep Zoom pipeline: which tiles are visible, what fallback tiles exist, and where tiles should be drawn on screen. Previously named SKDeepZoomTileScheduler; renamed to reflect its true purpose as a tile layout/geometry calculator rather than an async work scheduler.

SKDeepZoomViewport

Manages the Deep Zoom viewport — the logical window into the image. Handles coordinate conversions between element (screen), logical (0-1 normalized), and image pixel spaces. Mirrors Silverlight's MultiScaleImage viewport behavior.

Structs

SKDeepZoomDisplayRect

Represents a display rectangle in a sparse Deep Zoom Image. Defines a region of available pixels and the pyramid levels at which it is visible.

SKDeepZoomRectF

A floating-point axis-aligned rectangle used for screen-space tile geometry. Replaces anonymous tuples like (float SrcX, float SrcY, float SrcW, float SrcH) and SKRect usages in the SkiaSharp-agnostic DeepZoom pipeline.

SKDeepZoomRectI

An integer-precision axis-aligned rectangle used for tile pixel bounds within a pyramid level. Replaces anonymous tuples like (int X, int Y, int Width, int Height) throughout the DeepZoom pipeline.

SKDeepZoomTileId

Identifies a single tile in the pyramid.

SKDeepZoomTileRequest

A tile to be fetched, with a priority value (lower = higher priority).

SKDeepZoomViewportState

Immutable snapshot of viewport position and zoom.

Interfaces

ISKDeepZoomRenderer

Renders individual Deep Zoom tiles onto a drawing surface. Implement this interface to provide a custom rendering backend.

ISKDeepZoomTile

Opaque handle representing a decoded tile image. Concrete implementations (e.g. SKDeepZoomBitmapTile in SkiaSharp.Extended) wrap the actual decoded image in a rendering-backend-specific type.

ISKDeepZoomTileCache

Pluggable tile cache for the Deep Zoom rendering pipeline. Tiles are stored as ISKDeepZoomTile so that the cache is rendering-backend-agnostic.

ISKDeepZoomTileDecoder

Decodes raw image data from a stream into an ISKDeepZoomTile. Implement this interface to plug in a custom image decoder (e.g. SkiaSharp, ImageSharp).

ISKDeepZoomTileFetcher

Fetches tile data from a source (HTTP, file system, memory, etc.) and returns it as an opaque ISKDeepZoomTile. Implement this interface to plug in a custom tile source.

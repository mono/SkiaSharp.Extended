Pluggable renderer for Deep Zoom tiles. Implement this interface to customize how tiles are drawn onto the canvas (e.g., to add debug overlays, watermarks, or custom compositing).

public interface ISKDeepZoomRenderer : IDisposable
Inherited Members
IDisposable.Dispose()

Remarks

The default implementation is SKDeepZoomRenderer, which supports LOD fallback blending. Use the decorator pattern to wrap it with additional behaviour.

Methods
